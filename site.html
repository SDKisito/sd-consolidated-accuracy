À propos

Data Scientist : Un data scientist est un expert qui analyse, interprète et exploite les données pour résoudre des problèmes complexes. Grâce à des techniques avancées comme le machine learning et les statistiques, il transforme les données en informations stratégiques.

Ingénieur DevOps : Un ingénieur DevOps est responsable de l'automatisation, de l'intégration continue et de la gestion des infrastructures pour améliorer la productivité et la qualité des projets informatiques. Il facilite la collaboration entre les équipes de développement et d'opérations.

Compétences

  • Analyse des données, Machine Learning, Deep Learning
  • Programmation : Python, R, SQL
  • Gestion d'infrastructures cloud : AWS, Azure, Google Cloud
  • Outils DevOps : Docker, Kubernetes, Jenkins, CI/CD
  • Gestion de versions : Git, GitHub
  • Communication et visualisation : Tableau, Power BI, Matplotlib

Projets Réalisés

  • Analyse Prédictive des Ventes : Création d'un modèle prédictif pour une grande chaîne de distribution.
  • Automatisation CI/CD : Mise en place d'un pipeline DevOps pour une entreprise SaaS.
  • Système de Recommandation : Développement d'un algorithme pour personnaliser l'expérience utilisateur sur une plateforme de streaming.
